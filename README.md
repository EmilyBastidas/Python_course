# Curso de Python

Este repositorio contiene todos los ejercicios, proyectos y apuntes que estoy realizando durante mi aprendizaje de Python. El objetivo es documentar mi progreso y tener una referencia rápida de los conceptos clave.

## Estructura del Repositorio

- `inicio/`: Tu primer programa en Python
- `clases/`: Notas teóricas y ejemplos de cada sesión.
- `ejercicios/`: Retos y problemas resueltos.
- `proyectos/`: Pequeñas aplicaciones funcionales.
- `requirements.txt`: Librerías necesarias para ejecutar los scripts.

## Cómo ejecutar los scripts

1. Clona el repositorio:
   ```bash
   git clone https://github.com
   ```
2. Entra a la carpeta:
   ```bash
   cd nombre-del-repo
   ```
3. Ejecuta el archivo que desees:
   ```bash
   python nombre_del_archivo.py
   ```
